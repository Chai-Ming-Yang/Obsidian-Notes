#### **`IaaS "Infrastructure as a Service"`**
- More control over IT resources
	- Manage OS & Hardware (server , storage , resources)
- e.g. **AWS_EC2**/ Google Cloud
#### **`PaaS "Platform as a Service"`**
- **Deploy** software
- e.g. **AWS_EB** / Google App Engine
#### **`SaaS "Software as a Service"** 
- less control over IT resources
	- Ready to use software (end user app)
- e.g. Dropbox / Gmail

### Applications of Cloud
- Application Hosting (google docs)
- Backup Storage (online drive)
- Content Delivery (Netflix)
- Website
- Enterprise IT (Bank App, HR Portal)
- Database



### The Catch
- Shift from predictable, fixed capital expenses (CapEx) to variable operational expenses (OpEx)
- Strict cost governance
- Manage complex security key configuration
- Specialized skill requirement 
- Vendor lock-in
