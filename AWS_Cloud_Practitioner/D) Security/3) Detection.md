### Malware
|            |                                                                           |
| ---------- | ------------------------------------------------------------------------- |
| virus      | corrupt / delete data<br>self-propagate<br><br>                           |
| worm       | quick-spread (via network)<br>consume resource, install backdoors<br><br> |
| bot        | DDoS <br><br>                                                             |
| backdoor   | *(Trojan horse)*<br>steal info, enable intrusion<br><br>                  |
| Rootkit    | cloak, replace system file<br>difficult detection<br><br>                 |
| Spyware    | application embedded<br>non-official softwares<br><br>                    |
| Adware     | display advertisement (generate revenue)<br><br>                          |
| Ransomware | Lock resource/system<br>demand ransom payment<br><br>                     |

### Sources
- Untrusted Websites
- Removable Devices     *(USB)*
- Emails

### IDS *Intrusion Detection System*
"Alert upon threat detection"
Detection
- ==Anomaly-based==  -- compare **current traffic** with **Established Baseline**
- ==Signature-based== -- monitor & analyze traffic for **known patterns**
