## DRP *Disaster Recovery Plan*
restore business functionality (quick + minimal impact)
**RTO *Recovery Time Objective***
- MAX acceptable delay 
	- service interruption <> restoration
**TPO *Recovery Point Objective***
- MAX acceptable duration 
	- since last data recovery point
**WRT *Work Recovery Time***
- recover data + test process
- ensure system live for production
**MTD *Max Tolerable Downtime***
- MTD = RTO+ WRT
