# File Representation
1. Hardware
2. Directories
3. Commands
4. Processes
**Benefit: Interoperability between files   &   Transparency**

**File Names**
- case sensitive
- locally unique
- no spaces

**Extensions** (.txt)
- optional 

|          |                                             |
| -------- | ------------------------------------------- |
| `/`      | Root                                        |
| `/boot`  | Boot Files                                  |
| `/dev`   | Device                                      |
| `/etc`   | Config                                      |
| `/home`  | Standard User's home dir                    |
| `/media` | Removable Media                             |
| `/mnt`   | Network drives                              |
| `/root`  | Root user home dir                          |
| `/var`   | Log files \| print spool \| network service |
